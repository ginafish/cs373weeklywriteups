Forensics Basics

The most fundamental parts of computer forensics are making sure Key things: can you replicate it? Can you explain it? Yes you can analyze memory dumps! It's a core step. Only thing you are proving is what happened on the system.

When do you do forensic investigations? You do investigations in cases of fraud, intellectual property theft, hacker intrusions or data breaches, and more unnerving crimes like child exploitation and trafficking. Memory dumps are extremely useful for investigations, and there are other other tools like physically freezing the system that can be used to avoid data loss. With no data, there isn't anything to investigate. Memory dumps should be a core step of investigations.

Given how forensics is used in a court of law, it is important to understand what is evidence. Evidence is Evidence can be found from all kinds of levels: your network, operating system, applications that you use, peripherals, USB drives, and other human testimony.
